# 蓝桥杯嵌入式练习工程

蓝桥杯全国软件和信息技术专业人才大赛 - 嵌入式设计与开发赛道练习代码

## 硬件平台

- 开发板：CT117E-M4（蓝桥杯官方竞赛板）
- MCU：STM32G431RBT6
- IDE：Keil MDK-ARM

## 工程结构

每个工程目录结构如下：

```
XX-X/
├── APP/            # 应用层代码（LCD、LED、KEY、ADC等驱动封装）
├── Core/           # STM32CubeMX 生成的核心代码
├── Drivers/        # HAL 库驱动
├── MDK-ARM/        # Keil 工程文件
└── *.ioc           # STM32CubeMX 配置文件
```

## 使用方法

1. 使用 Keil MDK-ARM 打开 `MDK-ARM/*.uvprojx` 工程文件
2. 编译并下载到开发板

## 常用外设

- LCD：ST7735 液晶屏
- LED：8个可控LED灯
- 按键：4个独立按键
- ADC：模拟信号采集
- UART：串口通信
- RTC：实时时钟
- PWM：脉冲宽度调制
